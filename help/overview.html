Accident Mapper

Tool for mapping features of layer to features of another layer.

Displays links (to features of "layer 2") for one feature of "layer 1" at a time.

Can save and load csv.

Setting layers and fields

Before using the following will need specifying Layers:set layers.

Layer 1, and field to display/add to csv

Layer 2, and field to display/add to csv

Layer 3 and Field:optional: Only effect is to select features of layer 3 on map where field = displayed value when feature is selected in tool.

Selecting a feature

Features to edit links for can be selected through:

  • The drop down and arrows.
  • Selecting a feature on the layer and clicking 'From Layer'

Adding features:

Features can be added through:

  • "Edit:add selected features":
  • Edit:"add features within geometry" adds any features of layer 2 contained within the geometry of layer 1.
  • "Edit:add features within geometry to all": does this for every feature of layer 1

Removing Features:

Features can be removed through.

  • Right clicking on table and "drop selected rows".
  • Edit:clear:

Saving csv:

Save as csv through File:save as csv...

The resulting csv will have a header and 1 row per feature of layer 1 (with features of layer 2 mapped to it).

It will have 3 columns:

  • 1st column: Field of layer 1. Same name as field.
  • 2nd column: list of join field for mapped features of Layer 2. Same name as field.
  • 3rd column: Number of features in 2nd column. Named "Count".

Loading csv:

Load a csv through File-->load csv...

The csv will need to have a column with the same name as field 1 and a column with the same name as field 2.
