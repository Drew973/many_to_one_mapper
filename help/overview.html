Many to One Mapper

Tool for mapping many features of join layer to each feature of layer.

Can save and load csv.

Setting layers and fields

select the layer and field to use

select the layer to join to and field to add to csv

Selecting a feature

select a feature by:

  • selecting a feature on the layer and clicking 'From Layer'
  • entering a value into the box.

Mapping features of join layer to the selected feature

Right clicking the table gives options for editing which features of the join layer are mapped to the current feature.

  • Map selected features of the join layer to the current feature with 'Add selected features':
  • Add features within the geometry of the current feature through 'add features within geometry':
  • Remove features through 'drop selected rows':
  • Use 'select on layer' to zoom map to location of selected rows:

Tools:

  • Add features within geometry to all: does 'add features within geometry' for all features of layer.

The csv:

The purpose of the csv is to map features of the layer to features of the join layer through the join field and target field.

In order to do this these values must correspond to exactly one feature.

The csv will have a header and 1 row for each feature of the layer which has features of the join layer mapped to it.

csv has 3 columns

  • 1st column: target field of primary layer. Same name as field.
  • 2nd column: list of join field for mapped features of join layer. Same name as field.
  • 3rd column:named Count. Number of features in 2nd column.

Saving csv:

After selecting fields save as csv through File-->save as csv...

The plugin will check that each value corresponds to 1 feature when saving.

Loading csv:

Load a csv through File-->load csv...

The csv will need to have a column with the same name as the target field and a column with the same name as the join field.

Rows with errors will be skipped and produce a warning.
